# TistoryTalk(티스토리 톡)
- 메신저 기반의 티스토리 블로깅 아이폰 앱
- 제 13회 Daum DevDay2013 in jeju 출품작

#Project Goal 
- 티스토리 블로깅을 좀더 모바일기기에서 자유롭게 할수 있도록 한다.
